# Chess Engine

A personal chess engine and training tool to help improve at chess.

## Goals

- Build a chess engine capable of playing and analyzing games
- Analyze chess.com game history to identify weaknesses and patterns
- Provide targeted training recommendations

## Planned Features

- Chess engine (move generation, evaluation, search)
- chess.com game import and analysis
- Opening repertoire tracking
- Tactical pattern recognition
- Endgame training
